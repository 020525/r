方差分析的原理

在诸多领域的数量分析研究中，通常需要分析一种（或多种）因素是否能对目标结果造成影响。

例1：研究施肥量对农作物产量的影响。考虑三种施肥水平：10kg/亩、20kg/亩、40kg/亩；对每种施肥水平，随机选择10块实验田进行实验；在农作物收获时，收集每块实验田的亩产量，用方差分析就可以知道施肥量是否会影响农作物的产量。

例2：研究课后作业量对学生成绩的影响。考虑四种课后作业量：10题/天、20题/天、30题/天、40题/天；随机抽取四组学生，对每一组使用一种课后作业量；在一个月后，收集每个学生的成绩，用方差分析就可以知道课后作业量是否会影响成绩。

在这个实验中，除了课后作业量对最终成绩有影响之外，每个学生在实验之前的初始成绩也会影响最终成绩，因此在分析时应当把学生初始成绩的影响排除掉。

在方差分析中，我们将各种变量分为三类：

  • 因子（或称控制变量），即在实验中可以控制的自变量，例如施肥量、课后作业量等。在方差分析中，因子只能取若干个离散值，不能取连续取。例如，施肥量只能取10、20、40。因子的每个值都称为一个因子水平，例如施肥量有3个因子水平。在R中，因子必须是factor型变量；
  • 协变量。协变量也是实验中的自变量，但是协变量不能被实验人员控制，例如学生的初始成绩。在方差分析时，需要把协变量对因变量的影响剔除。在R中，协变量是连续型变量。
  • 因变量（或称观察变量），即实验的结果。例如，农作物的产量、学生的成绩等。因变量可以是连续型变量。


因子与协变量x1、x2、x3、…和因变量y的关系可以表示成一个函数：

y=f(x1, x2, x3, …)

在R的方差分析中，如果一个自变量是连续型(numeric)，那么它会被当成协变量；如果一个自变量是factor，那它会被当作因子。

根据因子和因变量的数量，方差分析可分为：

  • 单因素方差分析，即只有一个因子、一个因变量的方差分析；
  • 多因素方差分析，即有多个因子、一个因变量的方差分析；
  • 多元方差分析，即有一个或多个因子、多个因变量的方差分析。

单因素方差分析和多因素方差分析都属于一元方差分析。

方差分析的原理

方差分析，是用来检验两个或多个样本的均值是否有显著差别。在R的方差分析中，因子都必须是factor类型的数据（取离散值），因子将把因变量分为若干组（即若干个样本），每个组对应一个因子水平。

方差分析认为，如果一个因子能对因变量产生显著影响，那么在不同的因子水平下，因变量的值将有显著变动；反之，如果一个因子对因变量没有显著影响，那么在不同的因子水平下，因变量值的变动就不明显。

于是，方差分析的问题就转化为检验各因子在取不同水平时，因变量的平均值是否相等。在方差分析中，原假设是：在各因子水平下，因变量的平均值都相等，备择假设是：在某个因子水平下因变量的均值与其他因子水平下因变量的均值不相等。

方差分析假设因变量服从正态分布，并且各组的方差相等，即方差分析在比较几组均值时，假设抽得的几个样本都是来自正态总体，且有一个相同的方差，仅仅均值可以不相同。

ANOVA模型拟合

方差分析（ANalysis Of VAriance, ANOVA）与回归方法有一定联系，它们都是广义线性模型的特例。在R中，进行方差分析最常用的函数是：

result <- aov(formula, data=mydataframe)

其中，formula参数指定了各因子、协变量和因变量构成的公式，data参数指定了样本数据源，是一个数据框。在编写formula时，可以使用以下符号：

符号 用法
~ 分隔符号，左边为因变量，右边为自变量。例如：y ~ x1 + x2 + x3
+ 用于连接各自变量
: 表示自变量的交互项。例如：y ~ A + B + A:B
* 表示所有可能的展开项。例如：y ~ A * B * C 等价于 y ~ A + B + C + A:B + A:C + B:C + A:B:C
^ 表示交互项的最高次数。例如，y ~ (A+B+C)^2 等价于 y ~ A + B + C + A:B + A:C + B:C

所谓交互项是指两个或多个自变量的乘积，是公式中的非线性项。

公式中各自变量的顺序

在统计学中有三种常用的方差分析方法，它们处理各自变量的方式有区别。以公式

y ~ A + B + A:B

为例，三种方差分析方法分别为：

  • 类型I（序贯型），各变量的效应根据在公式中先后顺序做调整。对变量A不做调整，变量B根据变量A调整，交互项A:B根据变量A和B调整。这是aov()默认的方差分析方法。
  • 类型II（分层型），各变量的效应根据同层次和低层次变量的效应做调整。变量A和B在同一层次，它们都比交互项A:B层次低，因此变量A根据变量B调整，变量B根据变量A调整，交互项A:B同时根据变量A和B调整。
  • 类型III（边界型），每个效应根据模型其他各效应调整。变量A根据变量B和交互项A:B调整，变量B根据变量A和交互项A:B调整，交互项A:B根据变量A和B调整。这是SAS和SPSS默认使用的方差分析方法。

在各因子水平下，样本大小越不平衡，效应项的顺序对结果的影响越大。一般来说，越基础性的效应越需要放在公式前面，通常的顺序是：首先写协变量，之后是各个因子（越基础性的越放在前面），接着是双因素交互项，再接着是三因素的交互项，以此类推。

aov()使用序贯方差分析方法，如果想使用其他方差分析方法，可使用car::Anova()函数。

单因素方差分析

在单因素方差分析中，只有一个因子和一个因变量，样本将被每个因子水平分成一组，方差分析就是要检验各组的均值是否相等。

以multcomp包中的cholesterol数据集为例，这个数据集记录了50名患者接受降低胆固醇药物治疗的效果。在实验中，一共有5种治疗方法(trt)：使用传统药物20mg一天一次(记为1time)、10mg一天两次(记为2times)、5mg一天四次(记为4times)、使用新药物drugD(记为drugD)、使用新药物drugE(记为drugE)。

多重比较

在前面的例子中，aov()函数从整体上检验了各组患者的差异，告诉了这5组患者的疗效是有差异的，但并没有具体是哪组患者与其他患者有差异。

我们可以使用多重比较来解决这个问题，例如可以使用TukeyHSD()函数进行两两比较。

myfit <- aov(response ~ trt, data=cholesterol)

单因素协方差分析

在单因素协方差分析中，有一个因子，以及一个或多个协变量。

以multcomp包中的litter数据框为例，该数据框记录了药物对老鼠产崽的影响。在实验中，取74只怀孕的母鼠，对它们使用药物，之后记录下产崽时，每窝幼崽的数量和平均体重。

该数据框有4列：dose表示用药量(取值0, 5, 50, 500)，number表示一窝幼崽的数量，weight表示一窝幼崽的平均体重，gesttime表示怀孕时间长度。

在实验中，除了用药量(dose)会影响幼崽的体重(weight)之外，怀孕时长(gesttime)和一窝幼崽的数量(number)也会影响幼崽的体重。实验中，用药量(dose)是控制变量，作为因子，怀孕时长(gesttime)和一窝幼崽的数量(number)是协变量，幼崽的体重(weight)作为因变量。

双因素方差分析

在双因素方差分析中，存在两个因子。这两个因子除了各自影响因变量（主效应）之外，还会存在交互效应，因此在双因素方差分析的公式中，需要加上交互项。双因素方差分析一般写为：

myfit <- aov(y ~ x1 + x2 + x1:x2, data=mydataframe)

或者简写为：

myfit <- aov(y ~ x1*x2, data=mydataframe)

以基础安装包中的ToothGrowth数据框为例，该数据框记录了维生素C和喂食方式对海豚牙齿生长的影响。在实验中，分别对60只海豚喂食两种食物：橙汁(orange juice)或抗坏血酸维生素C(ascorbic acid)，在食物中的维生素C(Vitamin C)含量有三个水平：0.5mg、1mg或2mg，一段时间后记录下海豚牙齿的长度。

数据框ToothGrowth有3列：len表示海豚牙齿的长度，supp表示食物类型(VC或OJ)，dose表示VC含量(0.5, 1, 2)。supp是一个factor，可以直接放入方差分析，但dose是一个numeric vector，需要先把它转换成一个factor。

多元方差分析

多元方差分析中存在多个因变量，先将这些因变量组成一个向量，之后调用函数manova()，方法为：

myfit <- manova(formula, data=mydataframe)

以MASS包中的UScereal为例。该数据框随机抽取某零售商货架上放置的65种谷物食品，记录下食品的制造商(mfr)、在货架上在位置(shelf)、各营养成份(calories, protein, fat等)。

我们以UScereal数据框为例，检验不同制造商、不同货架层(shelf=1, 2, 3)上的食品，它们的营养成份(作为示例，我们只选取calories、fat和sugars三种营养成份)是否有区别，这是一个双因素三元方差分析过程。
